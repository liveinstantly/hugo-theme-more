Hugo More は Hugo 静的サイト ジェネレーターのテーマで、会社紹介、ドキュメント、ブログなど、複数の目的に適用可能な、シンプルでユニバーサルな Web サイトのために設計されています。

私たちの目的は、機能する Web サイトをできるだけ簡単に立ち上げて実行できるようにすることで、ユーザーにとって優れたコンテンツの作成に集中できるようにすることです。
